数字逻辑电路笔记

一、数制与码制

  1. 数制
  • 数制：十（D），2，4，8，16，R
  1. 数制的转换
  • R→十：
  • 十→2（4，8，16）
    • 整数：短除2至被除数为0，倒序排列
    • 小数：连乘2至结果为0，正序排列
  • 2→4（8，16）：每4（3，4）位组合转化
  1. 码制
  • 原码：正数：0… 负数：1… +2进制
    • 两个0
    • $-2^{n - 1} - 1\sim2^{n - 1} - 1$
  • 反码：负数1… +取反
    • 两个0
    • $-2^{n - 1} - 1\sim2^{n - 1} - 1$
  • 补码：
    • 负数反码+1
    • 只有1个0
    • $-2^{n - 1} \sim2^{n - 1} - 1$
  • 其它码：
    • BCD码：4位表示一位十位进制
      • 8421BCD：
      • 2421BCD：例12 = 00010010 = 00010010 = 01000101
      • 余3码：8421 + 3(0011) 每一位
      • 例12 = 00010010 = 00010010 = 01000101
    • BCD码计算：2进制加法（>9时+6(0110)）修正
    • 循环码/格雷码：减数取10的补码（进位— 无进位→求补/化10）
      • (00 01 11 10)
      • 二→格：高位相同 格（i）= 2（i）+ 2（i + 1）
      • 格→二：高位相同 2（i）= 格i + 2（i + 1）

二、逻辑函数及其简化

  1. 逻辑运算

    • 非：$\overline{S}$
    • 与（乘）：$S_1S_2$
    • 或（加）：$S_1 + S_2$
    • 与非：$\overline{S_1S_2} = \overline{S_1} + \overline{S_2}$
    • 或非：$\overline{S_1 + S_2} = \overline{S_1}*\overline{S_2}$
    • 异或：$S_1 \oplus S_2 = \overline{S_1}S_2 + S_1\overline{S_2}$
    • 同或（异或非）：$S_1 \odot S_2 = \overline{S_1}\overline{S_2} + S_1S_2$

  3. 完备集

  4. 基本规则

  • 代换：X替Y（式子）
  • 对偶（求对偶函数）：$0 \leftrightarrow 1，\cdot \leftrightarrow +$
  • 反演（求反函数）：$0 \leftrightarrow 1，\cdot \leftrightarrow +，X_i \leftrightarrow \overline{X_i}$
  1. 公式
  • 并项：$AB + \overline{A}B = B // (A + B)(\overline{A} + B) = B$
  • 消冗因子：$A + \overline{A}B = A + B // A(\overline{A} + B) = AB$
  • 消冗余项：$AB + \overline{A}C + BC = AB + \overline{A}C // (A + B)(\overline{A} + C)(B + C) = (A + B)(\overline{A} + C)$
  1. 绘制卡诺图
  • A - B 均为1形式，用圈连接
  1. 标准表达式
  • 最小项与标准与-或：$F = \sum m \cdot f_i$
  • 最大项与标准或-与：$F =\ prod M(f_i + M)$
  • 与-或-非表达式：
    • 反函数的与-或式取非（卡诺图的0）
    • 或-与-非两次非

五、奇偶校验与算术电路

  1. 奇偶校验
  • 奇偶校验电路：异或运算 $F = x_1 \oplus x_2 \oplus x_3 \dots$
  • 奇偶校验位（额外加一位，奇校验体内值使总1数为奇）
  • 74280（9位）：
    • EVEN 偶（1为偶时为1）
    • ODD 奇（1为奇时为1）
  1. 算术电路
  • 一位二进制（半加/全加）C_{in}：
    • 半：有低进位
    • 全：有低进位
  • 逐位进位全加器
  • 超前进位全加器：4位（74283）
  • 补码计算：
    • $A - B \Rightarrow A + (-B)$
    • $N_{补} = 2^n - N_{原} = N_{反} + 1$
    • $A - B = A + B_{补} - 2^n = A + B_{反} + 1 - 2^n$
    • $-2^n \Rightarrow 进位取反=借位$
  • $A - B > 0：进位处为1，即借0位$
  • $A - B < 0：进位处为0，即借1位，再求补得到结果$
